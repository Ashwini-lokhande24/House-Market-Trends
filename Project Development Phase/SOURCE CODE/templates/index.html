Housing Market Analysis

Visualizing Housing Market Trends using interactive dashboards and stories.

About the Project

This project analyzes housing market trends using Tableau visualizations. The dataset includes house sale prices, renovation years, number of bedrooms, bathrooms, floors, and house age.

Using interactive dashboards, the project helps identify patterns and factors affecting housing prices. The visualizations were created in Tableau and integrated into this web application.
